# 🌸 She Can Foundation — NGO Webpage

A simple, responsive single-page website built for the **She Can Foundation**,
created as part of the Frontend Development Internship selection task.

## 🔗 Live Demo
[Click here to view](file:///C:/Users/Swati/Downloads/shecan.html)

## 📋 Features
- NGO name and branding
- Hero section with heading and tagline
- About section with mission and impact stats
- Responsive image layout
- Call-to-action button
- Footer with contact info

## 🛠️ Built With
- HTML5
- CSS3 (no frameworks — pure CSS)

## 🚀 How to Run
Just open `shecan.html` in any browser. No setup needed.
